CLI RSS reader
================


Installation
------------
Create `.env` file
```dotenv
APP_ENV=prod
```

Install dependencies

``` bash
$ composer install
```

Usage
-----

``` bash
./bin/console

```


Testing
-------

``` bash
$ ./vendor/bin/phpunit
```
